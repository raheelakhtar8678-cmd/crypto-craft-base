# MyHarmony Discord Bot

## Project info

**MyHarmony** is an AI-powered Discord bot that provides smart moderation, community engagement, and server automation features.

## Features

- **AI Moderation**: Automatic detection of spam, toxic content, and rule violations
- **AI Engagement**: Polls, events, and interactive community features
- **Personality System**: Customizable AI personality for your server
- **Welcome System**: AI-generated personalized welcome messages
- **Leveling & Auto-Roles**: XP-based progression and automatic role assignment
- **Strike System**: Configurable warning, strike, and ban system
- **Logging**: Comprehensive moderation and activity logging

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I invite MyHarmony to my server?

1. Visit the Discord OAuth authorization page
2. Select your server
3. Configure the required permissions
4. Authorize the bot to join your server

## Quick Setup

Once the bot is in your server, use `/quicksetup` to instantly configure:
- Moderation channels and settings
- AI engagement features
- Welcome messages
- Logging configuration

## Commands

The bot supports various commands organized into categories:
- **Basic**: `/help`, `/ping`, `/info`
- **Settings**: `/settings`, `/quicksetup`, `/dashboard`
- **AI Features**: `/engagement`, `/personality_set`
- **Moderation**: `/ban`, `/kick`, `/warn`, `/purge`
- **Welcome**: `/welcome enable`, `/welcome channel`
- **Leveling**: `/rank`, `/leaderboard`
- **Entertainment**: `/8ball`, `/coinflip`, `/joke`

## Privacy & Security

MyHarmony processes only the minimum data required for functionality:
- Temporary message processing for moderation
- User IDs for strikes and leveling
- Server configuration data
- No permanent message storage (unless logging is enabled)
